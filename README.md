## **Project Overview**

This project is a **Bank Customer Transaction Analysis system**. It demonstrates how to analyze bank transactions using **Python**, **SQLite database**, and an **interactive Streamlit dashboard**.  

The system allows you to:  
- Track customer transactions  
- Analyze total amounts per customer  
- Identify most used services  
- Visualize monthly transaction trends  

This project is designed for **data analytics learning** and can be used as a portfolio project for beginners in **data analysis and visualization**.

---

## **Features**

- ✅ SQLite database backend to store transactions  
- ✅ Streamlit dashboard for interactive analysis  
- ✅ Filter by **Customer ID** and **Service Type**  
- ✅ Charts to visualize transaction trends and service usage  
- ✅ Easy to extend with more data or analytics
